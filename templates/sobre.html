O projeto foi criado com o objetivo de desenvolver uma IA simples utilizando tecnologias web e banco de dados.

Como o projeto funciona

Saiba mais como funciona o projeto
  • Recebe perguntas do usuário
  • Analisa a pergunta enviada
  • Procura respostas armazenadas no banco de dados
  • Responde automaticamente quando encontra uma resposta
  • Aprende novas informações com o tempo
  • Salva perguntas e respostas utilizando SQLite
  • Utiliza Python para processar as informações
  • Funciona através de uma interface web com HTML e CSS
